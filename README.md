# Newsweek
- kako pokrenuti

1. staviti cijeli folder s dokumentima u xampp/htdocs
2. pokrenuti apache i mySQL u xampp-u
3. u phpmyadmin staviti newsweek.sql (klikom na "import")
4. otvoriti http://localhost/newsweek/index.php

za administraciju - prijaviti se sa username: bvidovic, lozinka: lozinka
ili registrirati novi račun i promijeniti mu razinu s 0 na 1 u tablici "korisnici" (newsweek.sql) u phpmyadmin



SADRŽAJ:

index.php - naslovnica
kategorija.php - sve vijesti jedne kategorije
clanak.php - jedan članak
unos.php - forma za novu vijest
registracija.php - registracija korisnika
administracija.php - prijava + uređivanje i brisanje vijesti
style.css - dizajn stranice
newsweek.sql - sql s vijestima i korisnicima
